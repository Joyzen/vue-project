# element-starter

> 使用 vue-cli 脚手架构建使用 Element UI 框架的模板

## 开发环境

`NodeJs >= 6`

## 搭建项目

 - 可以直接使用 git clone 命令将此库克隆到您的本地git仓库
 - 或者直接下载压缩包到本地解压

 - 然后在项目目录下安装相关依赖插件
``` bash
npm install
```

## 开发模式

``` bash
# 启动本地服务，地址为 localhost:8010
npm run dev
```

## 打包发布项目

``` bash
# 构建一个生产包，在当前目录的 /dist 目录下
npm run build
```
